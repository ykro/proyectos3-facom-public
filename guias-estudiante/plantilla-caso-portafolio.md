# Plantilla — Caso de portafolio

Un buen caso de portafolio no muestra solo pantallas bonitas: cuenta una **historia**
con estructura **problema → proceso → resultado**. Tu app (por ejemplo "Bitácora
Visual") es a la vez un producto real y una pieza de tu portafolio creativo: doble valor.
Aquí lo que más suma es mostrar tu **proceso de desarrollo asistido por IA** (cómo
dirigiste a la IA), no solo el producto final.

> **Cómo usarla:** copia esta plantilla, borra los ejemplos y llénala con tu proyecto.
> Sirve tanto para tu portafolio web/LinkedIn como para preparar tu pitch del Demo Day.

## Checklist antes de dar por listo el caso

- [ ] Se entiende el **problema** y para quién es, en pocas líneas.
- [ ] El **proceso** muestra cómo dirigiste a la IA (con 1–2 prompts clave y capturas).
- [ ] El **resultado** tiene capturas finales limpias y profesionales.
- [ ] Cuento una historia (problema → proceso → resultado), no solo pantallas sueltas.
- [ ] Cuido accesibilidad y uso responsable de IA/imágenes (me presento como profesional serio).
